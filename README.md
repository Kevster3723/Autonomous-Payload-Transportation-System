# Autonomous Payload Transportation System

## Overview
This project is an autonomous robotic system designed to locate, pick up, and transport a payload to a designated platform without human intervention. The system integrates multiple sensors, embedded control, and omni-directional movement to achieve precise navigation and task execution.

The project was developed as part of a team-based engineering effort, with a focus on real-world system integration, reliability, and iterative debugging.

---

## Key Features
- Autonomous navigation and path control  
- Payload detection and transport mechanism  
- Omni-wheel drive for multi-directional movement  
- Multi-sensor integration for environment awareness  
- Real-time embedded control and decision-making

## System Architecture
The system consists of the following components:

- **Microcontroller:** Teensy 4.1  
- **Sensors:** Ultrasonic, Color, Infrared  
- **Actuators:** DC motors with omni-wheels  
- **Power System:** 12V Battery  
- **Control Logic:** Embedded C/C++ firmware handling navigation and decision-making  

---

## Technologies Used
- **Programming Languages:** C / C++  
- **Hardware Tools:** Arduino  
- **Debugging Tools:** Oscilloscope, multimeter, serial monitoring
