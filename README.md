Autonomous Payload Transportation System

This project was created for a class at the University of Denver. The goal of the project is to make an autonomous system with a Teensy4.1 and a budget of $30 to pickup and dropoff a payload.

The system has to detect the color of the payload and follow the corresponding line to the dropoff. It was also programmed to avoid obstacles.
The main components used were omni-wheels, 3 color sensors, an ultrasonic sensor, 2 infrared sensors, a servo, and 4 motors.
